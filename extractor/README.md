# IaC Extractor

## Supported IaC Files

| Name            | Extension(s)                  |
| --------------- | ----------------------------- |
| Terraform       | `.tf`, `.tfvars`, `.hcl`      |
| JSON            | `.json`, `.jsonl`, `.jsonc`   |
| YAML            | `.yaml`, `.yml`               |
| Container files | `Dockerfile`, `Containerfile` |
